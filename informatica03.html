CAPÍTULO 1: MICROSOFT EXCEL – FUNDAMENTOS E FUNÇÕES ESSENCIAIS

1.1 INTRODUÇÃO AO EXCEL

O Microsoft Excel é uma ferramenta indispensável no ambiente administrativo, sendo amplamente cobrada em concursos públicos por sua relevância no tratamento de dados, cálculos financeiros e organização de informações. Neste capítulo, abordaremos desde os conceitos mais básicos até situações complexas frequentemente exploradas pelas principais bancas examinadoras.

1.1.1 A Estrutura de uma Planilha Eletrônica

Uma planilha eletrônica é organizada em:

  • Linhas: identificadas por números (1, 2, 3...)
  • Colunas: identificadas por letras (A, B, C...)
  • Células: interseção de uma linha e uma coluna (ex: A1, B2, C3)

Cada célula pode conter:

  • Texto (dados alfanuméricos)
  • Números (valores numéricos)
  • Fórmulas (operações matemáticas e funções)
  • Datas e horas (tratadas como números especiais)
MACETE DE MEMORIZAÇÃO

"LINHA NÚMERO, COLUNA LETRA" – As linhas são numeradas (como degraus de escada) e as colunas são letradas (como um alfabeto vertical).

1.2 FÓRMULAS E OPERAÇÕES MATEMÁTICAS NO EXCEL

1.2.1 A Regra Fundamental: O Sinal de Igualdade (=)

Toda fórmula ou função no Excel OBRIGATORIAMENTE começa com o sinal de igualdade (=). Este é um dos pontos mais cobrados em provas e um dos erros mais comuns entre usuários iniciantes.

Exemplo incorreto: A1+B1 (sem o sinal de =) → O Excel interpretará como texto, não realizará o cálculo.

Exemplo correto: =A1+B1 → O Excel realizará a soma dos valores contidos nas células A1 e B1.

1.2.2 Operadores Matemáticos no Excel

OperadorFunçãoExemploResultado
+Adição=5+38
-Subtração=10-46
*Multiplicação=6*742
/Divisão=15/35
^Potenciação=2^38
ATENÇÃO ÀS PEGADINHAS DE PROVA!

As bancas frequentemente confundem os candidatos com os seguintes símbolos:

QUESTÃO TÍPICA DE CONCURSO:
"Para realizar uma multiplicação simples na célula C2, o usuário digitou #5*3 e pressionou Enter. Qual o resultado obtido?"

RESPOSTA: Nenhum cálculo será realizado. O Excel exibirá o conteúdo como texto, pois:
1. A fórmula não começa com =
2. O símbolo # não é reconhecido como operador matemático

1.2.3 Precedência de Operadores (Ordem dos Cálculos)

O Excel segue a ordem matemática padrão (PEMDAS):

  1. Parênteses
  2. Exponenciação (^)
  3. Multiplicação e Divisão (* e /)
  4. Adição e Subtração (+ e -)

Exemplo prático:

=2+3*4 → Primeiro 3*4=12, depois 2+12=14 =(2+3)*4 → Primeiro 2+3=5, depois 5*4=20

Exemplo de aplicação:

=A2*$B$1 → Ao arrastar para baixo, A2 vira A3, A4, mas $B$1 permanece fixo.

1.2.5 Estudo de Caso: Cálculo de Folha de Pagamento

Situação hipotética de prova

"Um servidor público precisa calcular o salário líquido dos funcionários onde:

  • Salário bruto está na coluna B
  • Desconto de INSS é 11% (fixo)
  • Desconto de IRRF segue tabela progressiva

Solução:

  • Para INSS: =B2*$G$1 (onde G1 contém 11%)
  • Para IRRF: usar função SE aninhada
  • Salário líquido: =B2-C2-D2 (bruto – INSS – IRRF)

PEGADINHA CLÁSSICA DE PROVA

QUESTÃO: "Um usuário utilizou a fórmula =PROCV(A2; D2:G100; 5; FALSO). Qual o resultado?"

RESPOSTA: #REF! – A matriz D2:G100 possui apenas 4 colunas (D, E, F, G), mas o índice solicitado é 5, ultrapassando o limite.

1.3.3 Função SE (Condicional)

A função SE é uma das mais versáteis e frequentemente combinada com outras funções.

Sintaxe:

=SE(teste_lógico; valor_se_verdadeiro; valor_se_falso)

Exemplo prático – Aprovação em Concurso:

=SE(B2>=70; "APROVADO"; "REPROVADO")

Se a nota em B2 for maior ou igual a 70, exibe "APROVADO"; caso contrário, "REPROVADO".

Funções SE Aninhadas (Múltiplas Condições):

=SE(B2>=90; "A"; SE(B2>=80; "B"; SE(B2>=70; "C"; "D")))

1.4.2 Filtros – Análise Seletiva de Dados

O filtro permite exibir apenas as linhas que atendem a critérios específicos, ocultando temporariamente as demais.

Características Importantes:

  1. NÃO apaga dados – apenas oculta visualmente
  2. Pode ser aplicado a qualquer tipo de dado (números, texto, datas)
  3. Permite múltiplos critérios simultâneos
  4. É reversível – basta remover o filtro
Tipo de DadoCritérios Possíveis

NúmeroMaior que, menor que, entre, igual a
DataAntes de, depois de, entre, este mês, este ano

ESTUDO DE CASO: Classificação de Servidores

Situação

Um órgão público possui uma planilha com todos os servidores e deseja:

  1. Classificar em ordem alfabética pelo nome
  2. Filtrar apenas servidores da Secretaria de Saúde

Procedimento Correto:

  1. Selecionar a planilha inteira (Ctrl+T)
  2. Guia Dados → Filtro (ou Ctrl+Shift+L)

  4. Na coluna "Secretaria", clicar na seta e desmarcar todas as opções, exceto "Saúde"

CAPÍTULO 3: E-MAIL – COMUNICAÇÃO PROFISSIONAL

3.1 CONCEITOS FUNDAMENTAIS

3.1.1 Campos de uma Mensagem de E-mail

CampoFunçãoVisibilidade
ParaDestinatários principaisTodos veem
Cc (Cópia)Destinatários em conhecimentoTodos veem
Cco (Cópia Oculta)Destinatários que devem ser ocultadosApenas o remetente sabe

3.1.2 Diferença Crucial: Cc vs. Cco

Cc (Com Cópia)
  • Todos os destinatários veem quem mais recebeu
  • Útil para comunicação em equipe
  • Transparente e colaborativo
Cco (Com Cópia Oculta)
  • Destinatários não veem os demais
  • Cada um vê apenas seu próprio endereço
  • Essencial para privacidade e envios em massa

ESTUDO DE CASO: Envio de Comunicado Oficial

Situação

Um diretor precisa enviar um comunicado para 100 funcionários, mas não deseja que os endereços de e-mail sejam expostos.

Procedimento Correto:

  1. Deixar os campos Para e Cc em branco
  2. Preencher apenas o campo Cco com todos os 100 endereços
  3. Escrever a mensagem normalmente
  4. Enviar

Resultado: Cada funcionário vê apenas seu próprio e-mail no campo Cco. O remetente tem a lista completa. Os destinatários não sabem quem mais recebeu.

⚠️ PEGADINHA DE PROVA

Afirmação FALSA:
"Para ocultar todos os destinatários, basta colocar os e-mails no campo Cco, e preencher o campo Para com o próprio e-mail."

CORREÇÃO: Para ocultar todos os destinatários, o correto é deixar Para e Cc em branco e colocar todos os endereços em Cco.

3.2.1 Limite Comum em Servidores

A maioria dos servidores de e-mail (incluindo órgãos públicos) possui limite de até 30MB por mensagem.

CAPÍTULO 4: REDES DE COMPUTADORES

4.1 PROTOCOLOS DE REDE

4.1.1 DHCP – Configuração Automática de Rede

DHCP (Dynamic Host Configuration Protocol) é o protocolo responsável por atribuir automaticamente:

  • Endereço IP
  • Máscara de sub-rede
  • Gateway padrão
  • Servidores DNS

Funcionamento:

  1. Computador solicita configuração (DHCP Discover)
  2. Servidor DHCP oferece um IP (DHCP Offer)
  3. Computador aceita (DHCP Request)
  4. Servidor confirma (DHCP Acknowledgment)

4.1.2 Comparação: Configuração Dinâmica vs. Estática

AspectoDinâmica (DHCP)Estática (Manual)
ConfiguraçãoAutomáticaManual
Ideal paraGrandes redesServidores, impressoras
Risco de conflitoBaixo (servidor gerencia)Alto (IP duplicado)
MobilidadeAlta (funciona em qualquer rede)Baixa (fixo)

4.2 DIAGNÓSTICO DE PROBLEMAS DE REDE

4.2.1 Método de Resolução de Problemas

Situação

Um site XYZ não responde, mas outro site ABC funciona perfeitamente.

Análise Lógica:

ObservaçãoConclusão
ABC funcionaRede local está ok
ABC funcionaConexão com internet está ok
XYZ não funcionaDNS pode estar ok (outros sites funcionam)
XYZ não funcionaServidor XYZ está com problema

RESPOSTA CORRETA: A conexão do website XYZ com a internet está interrompida.

4.2.2 Erros Comuns de Interpretação

Erro 1

"O problema é na minha rede local"
Correção: Se outros sites funcionam, a rede local está ok

Erro 2

"O DNS está com problema"
Correção: Se outros sites abrem, o DNS está funcionando

Erro 3

"Meu computador está com vírus"
Correção: Se apenas um site específico não funciona, é problema do site

CAPÍTULO 6: MICROSOFT WORD – FERRAMENTAS DE FORMATAÇÃO

6.1 PINCEL DE FORMATAÇÃO

6.1.1 O que é e Como Funciona

O Pincel de Formatação é uma ferramenta do Word que copia todo o conjunto de formatações de um texto para outro.

Formatações copiadas:

  • ✅ Tipo e tamanho da fonte
  • ✅ Negrito, itálico, sublinhado
  • ✅ Cor do texto e do fundo
  • ✅ Espaçamento entre linhas e parágrafos
  • ✅ Recuos (esquerda, direita, primeira linha)
  • ✅ Alinhamento
  • ✅ Marcadores e numeração

6.1.2 Como Usar

  1. Selecionar o texto com a formatação desejada
  2. Clicar no Pincel de Formatação (guia Página Inicial)
  3. Clicar no texto onde deseja aplicar a formatação

Dica: Clique duas vezes no pincel para aplicar a formatação em múltiplos locais.

6.1.3 O que o Pincel de Formatação NÃO COPIA

  • ❌ Conteúdo do texto (apenas a formatação)
  • ❌ Estilos de parágrafo (apenas formatação direta)
  • ❌ Imagens e objetos (apenas texto)

CAPÍTULO 7: MICROSOFT POWERPOINT – APRESENTAÇÕES

7.1 PAINEL DE ANOTAÇÕES

7.1.1 Função e Utilização

O Painel de Anotação (também chamado de Painel de Notas) permite que o apresentador insira:

  • Lembretes sobre cada slide
  • Roteiro da apresentação
  • Dados complementares que não devem aparecer para o público

Localização:

  • Parte inferior da janela do PowerPoint
  • Visível apenas no Modo de Exibição do Apresentador
  • Não aparece durante a apresentação normal

7.1.2 Comparação: Painel de Anotação vs. Comentários

RecursoVisibilidadePropósito
Painel de AnotaçãoApenas apresentadorRoteiro, lembretes
ComentáriosColaboradoresRevisão, feedback

7.1.3 Modo de Exibição do Apresentador

Recursos disponíveis:

  • Slide atual (visto pelo público)
  • Slide seguinte (prévia)
  • Painel de Anotação (privado)
  • Tempo decorrido
  • Ferramentas de navegação
  • Caneta/laser (para apontar no slide)

8.1 FORMATOS DE ÁUDIO

8.1.1 WAV – Padrão sem Compressão

WAV (Waveform Audio File Format) é um formato de áudio sem compressão (não comprimido).

Características:

  • ✅ Alta qualidade de áudio
  • ✅ Compatível com a maioria dos players

CAPÍTULO 10: LISTA DE VERIFICAÇÃO PARA PROVAS

10.1 PONTOS CRÍTICOS MAIS COBRADOS

Excel
  • ☐ Toda fórmula começa com =
  • ☐ Multiplicação usa * (não # ou @)
  • ☐ PROCV: entender todos os argumentos
  • ☐ Diferença entre FALSO e VERDADEIRO no PROCV
  • ☐ Validação de dados aceita números, texto e datas
  • ☐ Filtros podem ser aplicados a qualquer tipo de dado
  • ☐ Diferença entre Cc e Cco
Navegadores
  • ☐ Edge padrão = Bing, mas configurável
  • ☐ Safari é navegador (Apple)
  • ☐ Webmail não é navegador
  • ☐ URL ≠ servidor, protocolo ou ferramenta de busca
Segurança
  • ☐ Firewall: perfil público = mais restritivo
  • ☐ DHCP = configuração automática de IP
  • ☐ Gerenciador de Tarefas = monitoramento de desempenho
E-mail
  • ☐ Cco oculta todos os destinatários
  • ☐ Limite de anexo = 30MB (servidores comuns)
  • ☐ Copiar e-mail não dobra limite
Ferramentas
  • ☐ Pincel de Formatação copia todo o conjunto
  • ☐ Painel de Anotação é para o apresentador
  • ☐ WAV = áudio sem compressão

APÊNDICE: ANÁLISE DE QUESTÕES TÍPICAS

Questão 1 (Cesgranrio)

"Um servidor precisa classificar os dados da coluna C em ordem alfabética e, em seguida, somar os valores da coluna D. Qual o procedimento correto?"

Análise:

  1. Selecionar toda a planilha (dados relacionados)
  2. Guia Dados → Classificar (coluna C, A a Z)
  3. Para somar: =SOMA(D:D) ou selecionar intervalo
Questão 2 (FCC)

"Em uma mensagem de e-mail, o campo Cco deve ser utilizado quando:"

Análise:
Quando se deseja que os destinatários não vejam os endereços dos demais, mantendo a privacidade.

Questão 3 (Vunesp)

"No Windows 10, um computador conectado a uma rede de aeroporto deve utilizar qual perfil de firewall?"

Análise:
Perfil Público, por ser mais restritivo e seguro para ambientes não confiáveis.
